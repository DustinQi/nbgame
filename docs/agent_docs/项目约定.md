# 项目约定

> 本文件是 CLAUDE.md 的外链细则，记录**文件结构、提交格式、分支策略**等仓库约定。
> 触发条件：创建新文件、写提交信息、开分支时。

---

## 文件结构

```
Assets/
├── Scripts/          # C# 脚本
├── Scenes/           # 场景文件
├── Prefabs/          # 预制体
├── Resources/        # 运行时加载资源
└── Art/              # 美术资源（模型、贴图、材质）
```

---

## 提交信息格式

```
feat: 添加玩家移动功能
fix: 修复碰撞检测异常的 bug
refactor: 重构 UI 管理器代码
docs: 更新 README 说明
```

**类型说明**：
- `feat` — 新功能
- `fix` — 修复 bug
- `refactor` — 重构（不改变外部行为）
- `docs` — 文档变更

---

## 分支策略

- `main` — 稳定主线，可随时构建
- `feature/<功能名>` — 功能开发分支
- `fix/<问题描述>` — 修复分支
