# 王总，你好
## **欢迎你加入嘉悦nb_game项目的开发。你现在看到的窗口是一个叫Cursor的专业AI编码工具。**
### 但是别担心，你不需要写任何一行代码，写代码的工作由dustin和他的AI团队替你完成，你们的所有沟通都通过下方的窗口（“Terminal”）完成。

### 为了便于你快速熟悉本游戏开发的流程和步骤，以下是一些场景和指令供你参考

1. 如果你有时候不知道接下来该做什么、有点迷茫，想让dustin给你建议一下，你只需要输入：
**dustin我该干啥**

2. 如果你想针对某个问题（可以是技术、创意等任何问题）和dustin来一场讨论，你只需要输入：
**dustin咱们聊聊**

3. 如果你想让dustin帮你复盘一下当前项目的进展，你只需要输入：
**dustin向我汇报**

4. 如果你想开始做某个功能、让dustin帮你写代码，你只需要输入：
**dustin开工了**

5. 如果你写了一些代码后想让dustin帮你检查一下有没有问题，你只需要输入：
**dustin帮我检查一下**

6. 如果你运行游戏时遇到报错或者功能不正常、搞不定了，你只需要输入：
**dustin出问题了**

7. 如果你觉得做得差不多了，想让dustin把代码存起来备份一下（这样以后可以找回任何一版），你只需要输入：
**dustin保存一下**

8. 如果你遇到了AI工具或框架操作方面的问题，想找技术顾问dustin帮忙，你只需要输入：
**帮我找dustin**

9. 如果你想做一个复杂的功能，先让dustin帮你规划和设计一下方案，你只需要输入：
**dustin帮我规划一下**

10. 如果你做完了一个功能，想让dustin帮你收尾、更新进度，你只需要输入：
**dustin收工**

11. 如果你做完了一个功能，想在Unity里试一试好不好用，让dustin给你写一份详细的测试步骤，你只需要输入：
**dustin测试一下**

12. 如果你想让我更新一下项目进度文档，记录刚完成的工作，你只需要输入：
**dustin记录一下**
